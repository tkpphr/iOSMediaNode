# iOS MediaNode

## Overview
The tree node that has image and sound. 

## Installation

### CocoaPods
```
pod 'iOSMediaNode',:git => 'https://github.com/tkpphr/iOSMediaNode.git'
```

## Usage
1. Create the class that inherited "MediaNode" class.
2. Use it.

See details at a demo target.
